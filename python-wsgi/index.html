WSGI協定

  • Web Server Gateway Interface

  • 是python描述 web server 和 web application 的通信規範

  • 要做到wsgi, 必須有web server 和 web application

    • server

      負責接收 client的request,轉給 application，再將application 的response 轉回給 client

    • application

      接收由server轉發的request，處理requets並response。

      application中可以包含多個 middlewares，這些middleware 需要同時實現server與application，因此可以在WSGI server 和WSGI application 之間起調節作用。

      uwsgi協定

  • 也是種通信協議
    - uWSGI server的獨佔協定

  • 與 WSGI 不一樣
